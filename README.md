# 親方ドットコム LP

工事業者専門の集客支援サービス ランディングページ

## 技術スタック

- Vite + React + TypeScript
- Tailwind CSS v3
- Noto Sans JP（Google Fonts）

## 開発

```bash
npm install
npm run dev
```

## フォルダ構成

```
src/
├── sections/     # セクション単位のコンポーネント
│   ├── Header.tsx
│   ├── Hero.tsx
│   └── Section01.tsx（追加予定）
├── assets/
│   ├── images/   # 写真素材
│   └── icons/    # SVGアイコン
├── App.tsx
├── main.tsx
└── index.css
```
